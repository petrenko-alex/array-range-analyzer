**Функциональные тесты программы Array Range Analyzer**  
Запуск тестов осуществляется с помощью .bat файлов.
Каждый .bat файл содержит внутри описание проверяемой ситуации.

1. "Аргументы командной строки" - проверка поведения программы при задании разного количества аргументов командной строки.
2. "Считывание входных данных"  - проверка поведения программы при задании некорректных входных данных (ошибки чтения данных).
3. "Анализ без критических ситуаций" - проверка основной функции программы. Входные данные корректны, осуществляется анализ выражения.
4. "Анализ с возникновением критических ситуаций" - примеры ситуаций, когда помимо выходов за пределы массивов могут произойти критические ошибки
(например, присвоение переменной неопределенного элемента, т.е. элемента за пределами массива или зацикливание переменной)
