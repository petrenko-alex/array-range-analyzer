Папка содержит функциональные тесты программы Array Range Analyzer.
Запуск тестов осуществляется посредством запуска .bat файлов.
Каждый .bat файл содержит внутри описание проверяемой ситуации.

1) Папка "Аргументы командной строки" - проверка поведения программы при задании разного количества аргументов командной строки.
2) Папка "Считывание входных данных"  - проверка поведения программы при задании некорректных входных данных (ошибки чтения данных).
3) Папка "Анализ без критических ситуаций" - проверка основной функции программы. Входные данные корректны, осуществляется анализ выражения.
4) Папка "Анализ с возникновением критических ситуаций" - примеры ситуаций, когда помимо выходов за пределы массивов могут произойти критические ошибки
(например, присвоение переменной неопределенного элемента, т.е. элемента за пределами массива или зацикливание переменной)
